## Testing functionality

You need to create a _.env_ file with credential to your MongoDB Cluster and secret keyword for jwt-passport.

Like this:

```
MONGO_USERNAME=*****
MONGO_PASSWORD=*****
MONGO_DB=*****
MONGO_HOSTNAME=***.*****.mongodb.net
ACCESS_TOKEN_SECRET=****
```
